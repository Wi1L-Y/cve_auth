### description

This firmware uses R7WebsSecurityHandler to perform authentication. However, this firmware uses strstr to match filenames, which in this case may lead to authentication bypass. An attacker can append ?img/main-logo.png or ?reasy-ui-1.0.3.js to the URL to access any endpoint of httpd without authentication.


### Root case

file: /usr/sbin/httpd
